# Yndu Frontend

Vue 3 + TypeScript + Pinia frontend for the Yndu fresh produce delivery system.

## Tech Stack

- Vue 3 with Composition API
- TypeScript
- Pinia for state management
- Vite for build tooling
- GraphQL Code Generator for type-safe queries

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
